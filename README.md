# 🚢 Titanic Survival Prediction

## Overview
This machine learning project predicts whether a passenger survived the Titanic disaster using Logistic Regression.

## Dataset
The Titanic dataset contains passenger information such as:
- Age
- Gender
- Passenger Class
- Fare
- Number of Siblings/Spouses
- Number of Parents/Children
